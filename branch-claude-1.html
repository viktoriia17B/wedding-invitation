База: c336de9 (main) · 18 комітів · ~40 файлів, +700/−220 рядків · липень 2026. Мета гілки — довести односторінкове весільне запрошення (React 19 + Vite) до стану, готового до розсилки гостям. Пріоритет — виключно мобільна версія.

  • 7a80dde Workflow деплою перенесено з помилкового шляху src/github/workflows у робочий .github/workflows/deploy.yml — публікація на GitHub Pages тепер автоматична з пушу в main (джерело Pages треба перемкнути на «GitHub Actions»; npm run deploy через gh-pages лишився як легасі-шлях).
  • 3ed9ca2 README замінено з шаблонного Vite-тексту на реальні інструкції запуску проєкту.
  • f85f236 Скрипт npm run build:nginx — збірка з --base=/ для хостингу в корені домену (локальний nginx для тестів з телефона), тоді як стандартний build лишає base /wedding-invitation/ для Pages.

2. Фонова музика та сумісність аудіо

  • 050393c Доданий mp3-фолбек (src/assets/audio/hatrsyzy.mp3): Safari не грає ogg, тож useAudio обирає формат через canPlayType. Аудіо створюється ліниво після першої взаємодії (обхід autoplay-обмежень), кнопка перемикання з'являється після першого запуску.

3. Прев'ю посилання та захист форми

  • 77e95e8 OG-теги + public/og-image.jpg — у месенджерах посилання розгортається карткою з фото і описом. У форму доданий honeypot _gotcha (Formspree відсіює ботів). Рефи переведені на синтаксис React 19.

4. Архітектура скролу (найбільша зміна)

  • 2b624a4 Посекційний скрол: CSS scroll-snap (y mandatory + scroll-snap-stop: always) — один жест гортає рівно одну повноекранну секцію; хук useWheelSnap допомагає wheel-жестам на межах секцій.
  • 6b14ee1 Ключовий фікс тремтіння і перестрибувань на мобільних: скрол перенесено з документа у внутрішній контейнер .snap-scroller (html/body/#root — overflow: hidden). Документ не скролиться → мобільний браузер не ховає адресний рядок → висота в'юпорта стабільна → snap-точки не зсуваються і секції не «налазять» одна на одну. Секції — min-height: 100% контейнера замість viewport-одиниць. Програмні скроли (кнопка Hero, крапки навігації, wheel) ідуть через snapScrollTo() (src/utils/snapScroll.js), який на час smooth-анімації вимикає snap (інакше mandatory-snap б'ється з анімацією — ривки). У useWheelSnap — лок з продовженням на кожну інерційну подію: хвіст інерції трекпада не перегортає зайву секцію.

5. Візуальний поліш

  • 4b6ce27 Контраст тексту на фото (градієнтні скріми зверху/знизу), охайніший таймер зворотного відліку.
  • cab9c2a Плавна поява контенту секцій при першому вході у в'юпорт (useRevealSections на IntersectionObserver, замінив useScrollTrigger), крапкова навігація по секціях (DotNav), декоративні розділювачі (Divider), іконки.
  • c2c7c56 Секція локацій вміщається у в'юпорт малих телефонів (clamp-розміри від висоти екрана).
  • 2f0e016, abfd7f1, 964caaa Ітерації привітального тексту: більший кегль, золотий каліграфічний акцентний рядок — текст читається як звернення, а не примітка.

6. Типографіка

  • 8ce0a0f Елегантна весільна пара шрифтів замість Montserrat/Marck Script: EB Garamond (основний — заголовки, текст, форма; табличні цифри, щоб таймер не «сіпався») і Great Vibes (каліграфія — імена пари, акцентні рядки); Cormorant Garamond лишився для курсивних підзаголовків. Усі три покривають українську кирилицю. Шрифти самохостяться через @fontsource-пакети в бандлі — Google Fonts CDN блокується приватними браузерами/Pi-hole, і гості бачили б системні фолбеки.

7. Форма RSVP

  • 24770e7 Фікс поламаного відображення після відправки: FinalScreen рендерив вкладений <section> — той отримував власну snap-точку і колапсував по висоті. Тепер це <div> із flex: 1, який заповнює секцію; wheel-обробник враховує лише секції верхнього рівня.
  • 4123a71 Дружній текст помилки відправки з фолбеком «напишіть нам в особисті» — щоб заявка гостя не губилась мовчки при мережевих блокуваннях. (Сама причина помилки на тестовому стенді була серверною: CSP nginx без formspree.io у connect-src — виправлено в конфігурації сервера, поза цим репозиторієм.)
